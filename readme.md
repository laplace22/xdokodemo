xdokodemo
---------

## description

This software shares keyboard and mouse via network.
It can't be the client. Only server mode is implemented.
Use windows client dokodemo(*1).

## usage
* -p port  
     UDP port setting
* -f ipaddress[/range]  
     restrict ip address
* -x  
     dpms controling enable(default false)
